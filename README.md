# ForestOS
This is ForestOS an openrating system with forest theme - <br>
or actually just a website that acts like an OS (operating system). <br>
This project is for Tech nerds and forest enthusiasts.
Go check it out now at https://laup0403.github.io/ForestOS/

## Features:
-🪟 draggable, closable and openable windows <br>
-🕛 a taskbar with clock <br>
-🖼️ a live wallpaper <br>
-📝 Nutnotes, a notes app <br>
-☎️ Thincan Phone, an app, where you can <br>
 chat with animals <br>
-🥚 an extremly difficult to find easter egg <br>
-🐷 a pig <br>
